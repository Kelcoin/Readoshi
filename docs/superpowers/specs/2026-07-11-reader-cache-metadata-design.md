# 阅读器增强、智能缓存与元数据编辑设计

## 目标

在保持现有 LANraragi React Reader 操作习惯的前提下，加入可逐项关闭的阅读增强、带容量上限的图片缓存，以及完整的归档元数据编辑页面。

## 阅读器增强

- 阅读布局：单页、Webtoon、自动检测。
- 自动 Webtoon：抽样相邻页底部/顶部色带，排除近白色样本，按有效接缝的色差连续率判断；低置信度回退单页，并允许手动覆盖和重新检测。
- 双页、适应屏幕/宽度/高度/原始尺寸、裁白边、宽页拆分、宽页旋转均提供设置项。
- Webtoon 与双页互斥；宽页拆分与旋转互斥，拆分优先。
- 页码指示器模式移入阅读设置。自动模式基于实际图片渲染区域判断，任意至少 1 CSS 像素的交叠即隐藏，并加入安全边距和显示迟滞，避免闪烁。

## 智能缓存

- 自动上限为浏览器配额的 20%，限制在 256 MB 至 2 GB；也可手动选择 256 MB、512 MB、1 GB、2 GB。
- 90% 触发清理，清理至 75%。优先删除过期衍生图和最久未访问图片，保护当前归档、预加载任务与本次会话最近使用内容。
- IndexedDB 保存大小、归档、类型、变体和最后访问时间；Cache Storage 保存图片内容。应用外壳缓存不计入图片上限。
- 设置中展示用量、上限，并提供立即清理和全部清除。

## 元数据编辑

- 路由 `/metadata?id=<archive-id>`，可从归档菜单和阅读器进入。
- 文件名和 ID 只读；标题、摘要、标签可编辑。
- 标签支持芯片快捷删除、逗号/回车提交、粘贴多标签、退格删除最后一项、忽略大小写去重，并复用中文与拼音建议。
- 可运行 LANraragi 元数据插件；返回标签直接合并到当前表单，不预览、不覆盖现有未保存字段，也不自动保存。
- 保存时提交完整标题、摘要和标签。若服务器内容在编辑期间变化，阻止覆盖并提示重新载入。
- 删除复用现有“同步删除 EH 收藏夹”确认流程；页面同时提供开始阅读、保存和返回。

## 兼容与质量

- 新增功能默认采用保守值，旧设置可无损迁移。
- 核心策略和几何算法使用纯函数单元测试；构建、主题自检和差异检查必须通过。
- 不引入新的运行时依赖。
